France-Increment
================

Textmate Bundle Commands to increment numbers

Usage
------

Select numbers and increment via:

<table>
    <tr>
        <td>+1</td>
        <td>Control &uarr;</td>
    </tr>
    <tr>
        <td>-1</td>
        <td>Control &darr;</td>
    </tr>
    <tr>
        <td>+10</td>
        <td>Control Shift &uarr;</td>
    </tr>
    <tr>
        <td>-10</td>
        <td>Control Shift &darr;</td>
    </tr>
</table>

Installation
------------

**Textmate**

1. **Download** as ZIP
2. Expand ZIP file
3. Double-click **.tmbundle**
